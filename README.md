# IPL Data Analysis Project

This repository contains a comprehensive data analysis of the Indian Premier League (IPL) cricket matches and deliveries. The project involves data extraction, cleaning, exploratory data analysis (EDA), and creating strategic dashboard ideas using Power BI.

## Project Structure

*   **`IPL_Analysis.ipynb`**: High-level Jupyter Notebook containing exploratory data analysis (EDA) using pandas, matplotlib, and seaborn. It explores trends, player metrics, and team performances.
*   **`PowerBI_Dashboard_Ideas.md`**: A detailed, step-by-step workflow guide containing DAX formulas and data modeling outlines for building an interactive Power BI dashboard.
*   **Data Files**:
    *   `matches.csv` & `deliveries.csv`: Raw IPL data.
    *   `cleaned_matches.csv` & `cleaned_deliveries.csv`: Data cleaned and pre-processed for analysis and Power BI importing.

## Objective

1.  **Exploratory Data Analysis**: Clean the dataset and analyze team performance, player statistics (batsman runs, bowler economy, boundaries), and match outcomes from various seasons of the IPL.
2.  **Visualization & Dashboarding**: Structure the data specifically to integrate into Power BI to create dynamic visualizations that surface strategic insights like toss impact, player analytics, and match venue distributions.

## How to used

*   **Python/Jupyter**: Run `IPL_Analysis.ipynb` locally using Jupyter Notebook or JupyterLab to follow the data transformation methodology.
*   **Power BI**: 
    1. Import the `cleaned_*.csv` datasets into Power BI.
    2. Follow the steps outlined in `PowerBI_Dashboard_Ideas.md` to define relationships, build DAX measures, and structure the visuals.
